# Dolmenwood
This is a Foundry VTT game system implementing the Dolmenwood ruleset. It can be used together with the dolmenwood-foundry-content-conversion module to fully enjoy Dolmenwood in Foundry VTT.

The system is still in development, but most major features concerning player character sheet are already implemented and automated.
